Library Application

This is a library application that allows users to manage a collection of books. Users can add new books using a form, display them on the page, update their read status, and remove them from the collection. All book data is stored in an array and dynamically managed with JavaScript.
Knowledge Used: JavaScript & DOM Manipulation,
HTML, CSS

Library Application

This library application is a dynamic web app built to manage a personal book collection. It is utilizing an array to store book objects and a form for users to input book details like author, title, number of pages, and read status. The form appears with a "NEW BOOK" button, and JavaScript handles the form submission, preventing default behavior and adding the book to the array using an object constructor.

A display function loops through the array to show each book on the webpage as individual cards. Each book display includes buttons to remove the book from the library and to toggle its read status. This is by using DOM elements associated to the book objects for easy manipulation.

The project focuses on client-side functionality, allowing users to manage their library during their session without persistent storage. This showcases my ability to perform basic CRUD operations and DOM manipulation, laying the groundwork for future enhancements such as storage of the array in order to save user input from session to session.

Calculator

For this project, I applied my knowledge of JavaScript, HTML, and CSS to develop a fully functional on-screen calculator. I implemented custom functions for addition, subtraction, multiplication, and division to ensure accurate and efficient calculations. To enhance the user experience, I designed a user-friendly HTML interface with buttons for digits, operators, an equals key, a clear button, and a display area, using CSS to improve the visual appeal and clearly distinguish between numbers and operators.

To ensure secure and robust expression evaluation, I avoided risky functions like eval() and built my own methods. I created dynamic display functions to provide real-time feedback and implemented logic to handle issues like division by zero, multiple decimal points, and incomplete inputs. Additionally, I added a backspace button for corrections and rounded decimal results to prevent overflow, enhancing the calculator's usability.

This project helped me solidify my understanding of fundamental web development concepts and demonstrated my ability to create interactive, reliable, and visually appealing web applications.

Etch-A-Sketch

In this project, I set out to build an interactive web-based sketchpad inspired by the classic Etch-A-Sketch toy, aiming to practice and showcase my DOM manipulation and Flexbox layout skills. My objective was to create a browser version of a sketchpad using JavaScript and Flexbox.

I used JavaScript to dynamically generate a 16x16 grid of square div elements within a container div. I applied Flexbox to display them as a grid, ensuring a clean and responsive design. For the drawing feature, I used a hover effect that changes the color of the grid squares as the mouse passes over them, allowing users to draw. Additionally, I added a button that lets the user resize the grid, clearing and resizing the grid accordingly while keeping a total width.

I enhanced the project by making each grid square change to a random color upon interaction and implementing a feature where each interaction with a square darkens its color by 10%, achieving a full color square after ten interactions.

This project not only highlights my technical skills but also demonstrates my ability to solve problems, research new methods, and create engaging user experiences.

Cookbook Website

A full stack personal project utilizing React and Node to create, read, update, and delete recipes and their users in databases.
Knowledge Used: ReactJS, NodeJS, MySQL, Insomnia, HTML, CSS

Cookbook Website

This project was started because I wanted to gain more experience with the ReactJS and NodeJS technologies. There are many recipes utilized in my household and I gained inspiration for this when I find myself hunting through cookbooks, saved web pages, and anything else just to find something that was made a few weeks ago. I wanted to have a central location to digitize and hold my favorite meals.

The majority of the project was focused on the implementation and functionalities of features and the design elements followed after. I wanted to make sure the databases were able to be used to create, read, update, and delete everything that is needed when it comes to cooking and recipes. I plan to make this an ongoing project of mine and have already implemented some features such as user login/logout, account registration, like/unlike buttons and counter, profile pages, password change, and recipe edit/delete (if it is your recipe). I have plans for many more features and hope to keep updating and bettering this project of mine so others may potentially find it useful as well.

I Am Brian Teeters

Growing up art had a significant influence on my life. I found that I could get lost for hours experimenting with colors, patterns, and shapes and pairing them together in unexpected ways. Through this, my career path as a graphic designer started.

After having been in the field for a number of years, I began to want something involving more of a challenge. I wanted to engage both sides of my brain, logical and creative. During college I had some classes dealing with web design and coding and enjoyed those thoroughly. Realizing this I decided to persue a degree in software development to expand my skill set. Keeping a creative outlet has always been important to me and the career shift allows me to add the challenge and logical engagement I have been looking for.

As a developer, designer, and artist I always enjoy being able to insert a part of my personality into each project. My goal is to open the door for an interaction with the user. I create with the purpose of impacting the lives of those around me with my work as well as meeting the needs of the end-user, and client alike through clear and well thought out design.
