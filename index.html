Alien Invasion Game

A personal project utilizing Python and the Pygame library mimicking the basic gameplay of the classic "Space Invaders" retro game.

Alien Invasion Game

I enjoyed working through this project as I have always played video games. Cookbook Website

A full stack personal project utilizing React and Node to create, read, update, and delete recipes and their users in databases.
Knowledge Used: ReactJS, NodeJS, MySQL, Insomnia, HTML, CSS

Cookbook Website

This project was started because I wanted to gain more experience with the ReactJS and NodeJS technologies. There are many recipes utilized in my household and I gained inspiration for this when I find myself hunting through cookbooks, saved web pages, and anything else just to find something that was made a few weeks ago. I wanted to have a central location to digitize and hold my favorite meals.

The majority of the project was focused on the implementation and functionalities of features and the design elements followed after. I wanted to make sure the databases were able to be used to create, read, update, and delete everything that is needed when it comes to cooking and recipes. I decided to utilize the React and Node full stack because it seems to be one of the more popular and widely used languages/technologies for this type of task.

I plan to make this an ongoing project of mine and have already implemented some features such as user login/logout, account registration, like/unlike buttons and counter, profile pages, password change, and recipe edit/delete (if it is your recipe). I have plans for many more features and hope to keep updating and bettering this project of mine so others may potentially find it useful as well.

The code can be found using the GitHub icon on the main page or by clicking here.

Tough Mudder Mobile UX/UI Redesign

I wanted to make the Tough Mudder website easier to navigate. I prioritized making links obvious as to where it would lead and the information that would be available on the page.
Knowledge Used: Adobe Xd, UX/UI Practices

Tough Mudder Mobile UX/UI Redesign

This class project idea came from the repeated, personal, use of the Tough Mudder website. I have had trouble finding specific information on races, events, or just Tough Mudder news in the past. With this in mind, I wanted to make the information that is currently available on the website more easily found as some of the links were not obvious or just did not lead to where the user expected.

The task was to only create a mobile version and unfortunately that is all that the course time allowed for. However, I was able to implement hover states and various other things that would be more geared for the tablet and desktop variations.

I Am Brian Teeters

Growing up art had a significant influence on my life. I found that I could get lost for hours experimenting with colors, patterns, and shapes and pairing them together in unexpected ways. Through this, my career path as a graphic designer started.

After having been in the field for a number of years, I began to want something involving more of a challenge. I wanted to engage both sides of my brain, logical and creative. During college I had some classes dealing with web design and coding and enjoyed those thoroughly. Realizing this I decided to persue a degree in software development to expand my skill set. Keeping a creative outlet has always been important to me and the career shift allows me to add the challenge and logical engagement I have been looking for.

As a developer, designer, and artist I always enjoy being able to insert a part of my personality into each project. My goal is to open the door for an interaction with the user. I create with the purpose of impacting the lives of those around me with my work as well as meeting the needs of the end-user, and client alike through clear and well thought out design.
